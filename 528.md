
# Macで、OSの言語設定とは別にOfficeソフト(Word, Excel, Powerpoint)の言語設定を変更する
# 背景
Macの言語設定を英語にしているが、フォントの選択画面のプレビューなど日本語で表示した方が嬉しい場面が多い。Windows版のOfficeソフトはFile>Optionからソフトの表示言語を変更できるが、Mac版のOfficeソフトはOSの設定言語を表示言語に反映し、ソフトには表示言語を変更するUIがない。そこで調べるとMacでもコマンドラインから表示言語を変更できることを知ったため日本語記事として共有する。

# 手法
- Terminalで以下のコマンドを実行する
```
$ defaults write com.microsoft.Powerpoint AppleLanguages '(ja)'
$ defaults write com.microsoft.Word AppleLanguages '(ja)'
$ defaults write com.microsoft.Excel AppleLanguages '(ja)'
```
- Activity Monitorでプロセスを終了して再起動すると言語が反映されている(ウィンドウを閉じて再起動するだけでは反映されない)
