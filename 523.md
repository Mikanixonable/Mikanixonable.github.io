## Java始め方(windows11)
Javaの環境構築は以下の手順でできる

- Java公式サイト[https://www.oracle.com/jp/java/]右上の「javaをダウンロード」からWindows x64 Installerをダウンロードする。
- ダウンロードしたインストーラーをインストールする。このさいインストール先(「C:Program Files/Java/jdk-15」みたいなもの)をコピーしてメモする。
- Windowsの設定>システム>システムの詳細設定>環境変数　のPathを選択し　編集>新規　から　インストール先のbinフォルダ(「C:Program Files/Java/jdk-15/bin」みたいなもの)を追加する。

## Java使い方
Javaのコマンドラインからのコンパイルと実行は以下の手順でできる。

- 拡張子javaのテキストファイルを作る。(拡張子表示設定をONにしたエクスプローラー上で右クリック>新規作成>テキスト ドキュメントでテキストファイルを作り、ファイル名を「Test.java」みたいに変えて拡張子をjavaに変更する)
- javaのソースコードをテキストファイルに書く。ただし、先頭行のクラス名はファイル名と同一にする。(ファイル名がTest.javaなら先頭行はpublic class Test {となる)
- エクスプローラー上で右クリック>ターミナルで開く　で開いたターミナルで
```javac Test.java```
と入力しエンターキーを押すことでソースコードが拡張子classのファイルにコンパイルされる。
- ターミナルで
```java Test```
と入力しエンターキーを押すことでclassファイルが実行され標準出力(ターミナルの画面)に実行結果が表示される。ゲームなどGUIのプログラムの場合は別のウィンドウが開く。

## Javaコード例

FizzBuzz
```java
public class Test {
    public static String fizzbuzz(int n){
        if (n % 15 == 0){
            return "FizzBuzz";
        }
        if (n % 3 == 0){
            return "Fizz";
        }
        if (n % 5 == 0){
            return "Buzz";
        }
        return String.valueOf(n);
    }

    public static void main(String[] args){
        for (int i=1; i <=100; i++){
            System.out.println(fizzbuzz(i));
        }
    }
}
```

階乗
```java
public class Test {
    public static int factorial(int n){
        if (n <= 1){
            return 1;
        }
        return n * factorial(n - 1);
    }

    public static void main(String[] args){
        System.out.println(factorial(5));
    }
}
```
フィボナッチ
```java
public class Test {
    public static int fibo(int n){
        if (n <= 1){
            return n;
        }
        return fibo(n-2) + fibo(n-1);
    }

    public static void main(String[] args){
        for (int i = 1;i <= 10; i++){
            System.out.println(fibo(i));
        }
    }
}
```

世界のナベアツ
```java
public class Test {
    public static String nabeatsu(int n){
        if (n % 3 == 0) {

    }

```


## Javaとは

90年代に誕生したC系の改良言語であり、マルチプラットフォームが特徴。Minecraft等に使われている。
70年代のC、80年代のC++よりは新しく、2000年代のC#よりは古い。CやC++よりはシンプルで整理されているが、C#よりは複雑な書き方をする場合がある。またJavaScriptはJava人気にあやかって命名された全然関係ない別の言語である。
